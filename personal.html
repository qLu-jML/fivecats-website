📋 How to Add a New Report

  1. Upload your HTML or PDF file to Firebase (e.g. reports/my-report.html)
  2. Open personal.html in a text editor
  3. Find the DOCUMENTS array near the bottom of the <script> tag
  4. Add a new entry following the same format as the examples
  5. Run firebase deploy to push the updated page
